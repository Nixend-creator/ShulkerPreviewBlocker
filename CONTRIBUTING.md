# Contributing

Thank you for considering contributing to ShulkerPreviewBlocker! Pull requests are welcome. For major changes, please open an issue first to discuss what you would like to change.

## Code Style

Please try to follow the existing code style (indentation, naming conventions, etc.).

## Pull Request Process

1. Fork the repository.
2. Create a new branch (`git checkout -b feature/AmazingFeature`).
3. Make your changes.
4. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
5. Push to the branch (`git push origin feature/AmazingFeature`).
6. Open a pull request.
